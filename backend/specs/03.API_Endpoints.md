The endpoints for API are:

## For user authentication and authorization:
POST /api/v1/auth/register
POST /api/v1/auth/login
POST /api/v1/auth/logout
POST /api/v1/auth/unsubscribe


## For transactions (*):
Create transaction:             POST /api/v1/transactions/
Get all transaction of user:    GET /api/v1/transactions/
Get filtered transaction of user:    GET /api/v1/transactions/q?
Get single transaction of user: GET /api/v1/transactions/:id

(*) Once transaction is created cannot be deleted.

## For Budget:
Create budget:                      POST /api/v1/budget/
Create BudgetCategory:              POST /api/vi/budget/:id/category    It must be possible to add a category (in incomes or expenses) once the budget is already created
Create BudgetGroup:                 POST /api/vi/budget/:id/group       It must be possible to add a group (in a specific incomes or expenses category) once the budget and BudgetCategory are already created
Create BudgetItem:                  POST /api/vi/budget/:id/item        It must be possible to add a item (in a specific incomes or expenses group) once the budget and BudgetCategory and BudgetGroup are already created

Get budget details for user:        GET /api/v1/budget/:id
Get budget transactions for user:   GET /api/v1/budget/:id/transactions

Update budget details for user:     PATCH /api/v1/budget/:id
Update BudgetCategory:              PATCH /api/vi/budget/:id/category    It must be possible to update a category (in incomes or expenses) once the budget is already created
Update BudgetGroup:                 PATCH /api/vi/budget/:id/group       It must be possible to update a group (in a specific incomes or expenses category) once the budget and BudgetCategory are already created
Update BudgetItem:                  PATCH /api/vi/budget/:id/item        It must be possible to update a item (in a specific incomes or expenses group) once the budget and BudgetCategory and BudgetGroup are already created

Delete budget for user:             DELETE /api/v1/budget/:id
Delete BudgetCategory:              DELETE /api/vi/budget/:id/category    It must be possible to delete a category (in incomes or expenses). All asociated groups, items and buckets must be deleted
Delete BudgetGroup:                 DELETE /api/vi/budget/:id/group       It must be possible to delete a group (in specific incomes or expenses category). All asociated items and buckets must be deleted
Delete BudgetItem:                  DELETE /api/vi/budget/:id/item        It must be possible to delete a item (in specific incomes or expenses group). All asociated buckets must be deleted.
Also, associated transactions to deleted items must be deleted.
