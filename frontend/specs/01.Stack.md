The stack for using in the Budgeteer Frontend is:
- Frontend: Developed in react.js version 19 + vite 8.0 (stable version) + typescript 5.x (stable version)  + Tanstack Router for page routing and Tanstack Query (if best option) + CSS Classes

Also use the follwing for implementation of Budget Creation alternatives (detailes in use cases document):
- shadcn/ui Stepper — for the step-by-step navigation (for budget creation)
- React Hook Form + Zod — for validation per step (we already use Zod)
- dnd-kit — for the hierarchy builder step (drag to nest budget categories, budget groups, budget items)
